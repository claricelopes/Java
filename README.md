# Exercícios de POO em Java

Repositório com os exercícios propostos na disciplina de **Programação Orientada a Objetos (POO)**, desenvolvidos em **Java** como prática dos conteúdos vistos em aula.

## Objetivo
Este repositório tem como objetivo armazenar e organizar os exercícios passados pelo professor durante a disciplina, acompanhando minha evolução no aprendizado de POO.

## Tecnologias
- Java
- JDK (versão utilizada no ambiente local)

## Conteúdo
Os exercícios podem abordar temas como:
- Classes e Objetos
- Métodos
- Atributos
- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Construtores
- Coleções e outros conceitos de Java
